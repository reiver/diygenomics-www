DIY Genomics

This is a site to help those interested in doing DIY genomics.

Getting Started

When people first get into DIY genomics, they often start with their own DNA data. So, for exampe, they have already had their DNA scanned using something like 23andMe, FamilyTreeDNA, or others; have downloaded their raw DNA file; and have started off by using the various tools out there, which you can apply to your DNA.

Some of the tools that can be used include:

  • Promethease
  • Interpretome
  • DIY Dodecad
  • Dodecad Oracle
  • HarappaWorld
  • HarappaWorld Oracle

(Some of these tools are also available online at GEDmatch.)

In addition to this, people will also sometimes join some DIY genomics projects out there. Such as:

  • Dodecad Ancestry Project,
  • Harappa Ancestry Project
  • Eurogenes Genetic Ancestry Project
  • African Ancestry Project
  • Magnus Ducatus Lituaniae Project
  • Fennoscandia Biographic Project
  • Malagasy Ancestry Project

Getting Sophisticated

After people run their DNA through some or all those tools, they can a bit more sophisticated in the DIY genomics endeavors.

Typically, this sophisticated starts with plink.
